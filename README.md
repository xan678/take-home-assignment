# Django Assignment

## Technologies used :
- React + TypeScript
- Redux + Redux Persist
- Django Rest Framework + Redis


## Steps to run :

1. ``` 
    pip install requirements.txt
    ```

2. Set up REDIS_URL in .env file
    

3. ```
    cd ./github-search-ui

    npm install

    npm run dev

    ```

4. ```
    cd ..

    python manage.py runserver
    ```
